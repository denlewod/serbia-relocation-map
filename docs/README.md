# Documentation

This folder contains technical documentation for the Serbia Relocation Map project.

## Structure

| Path | Purpose |
|---|---|
| [`architecture/decisions/`](./architecture/decisions/) | Architecture Decision Records (ADRs) — historical log of key technical decisions and their rationale |

## Other documentation

Some documents live outside this folder:

- **Vision & Scope** — [Google Docs](https://docs.google.com/document/d/1qZQrRfsqElZusRKWUQFgk9zrYkP_TsOiGwWpL12R6UM/edit) (the source of truth for business context)

- **Issue tracker** — [GitHub Issues](https://github.com/denlewod/serbia-relocation-map/issues)

## Conventions

- All technical docs in this folder are written in **Markdown** (`.md`)
- Business-level documents (Vision & Scope, presentations, etc.) live in Google Drive
- Each significant architectural decision is captured as a new ADR (see `architecture/decisions/`)
